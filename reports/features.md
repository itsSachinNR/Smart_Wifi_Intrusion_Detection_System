# Features

## 1. Device Scanner
Scans all devices connected to the WiFi network using ARP requests.

## 2. Authorized Devices
Stores known devices and compares them with detected devices.

## 3. Intrusion Detection
Detects unknown devices and flags them as intrusions.

## 4. Alerts
Notifies the user when an unauthorized device is detected.

## 5. Logging
Stores all device activity and intrusion events.
