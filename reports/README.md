# Smart WiFi Intrusion Detection System

## 📌 Project Overview
This project detects unauthorized devices connected to a WiFi network and alerts the user.

## 🚀 Features
- Device scanning
- Intrusion detection
- Alert system
- Logging

## ⚙️ Technologies Used
- Python
- Scapy
- Flask

## 📂 Project Structure
- core/ → detection logic  
- scripts/ → scanning and alerts  
- data/ → logs  
- reports/ → documentation
